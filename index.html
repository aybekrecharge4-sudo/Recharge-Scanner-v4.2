How Scoring Works

Each source's raw scores are normalized to 0-100 using per-source floor/ceiling ranges so they can be compared fairly. Then a weighted composite score is calculated: Tier 1 sources (Oxylabs, Trends, News, Reddit) carry ~50% of the weight, Tier 2 (Steam, Epic, YouTube, Competitors) ~30%, and Tier 3 (deal sites, anime, wiki) ~20%. Items appearing in multiple sources get a confidence multiplier (up to 1.0x for 4+ sources), and a freshness factor boosts very recent items.
